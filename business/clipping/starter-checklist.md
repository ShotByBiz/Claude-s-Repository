# Clipping — Same-Day Starter Checklist

Run this top to bottom. Most of it is free. Goal: first clips posted today, first
payout-eligible views this week.

## Setup (≈1 hour, one time)
- [ ] Create posting accounts: TikTok, Instagram (Reels), YouTube (Shorts),
      Facebook (Reels). Faceless is fine — pick a simple niche handle.
- [ ] Sign up for a clipping platform: **Vyro** and/or **ClipAffiliates**.
- [ ] Sign up for an AI clip tool: **OpusClip** or **Ssemble** (free tier to start).
- [ ] Pick **one niche** you'll actually enjoy (gaming, podcasts, motivation,
      finance, comedy). Niche focus > random clips.

## First batch (≈30–45 min)
- [ ] Choose a campaign whose content has clear hooks.
- [ ] Paste the source video URL into your clip tool; generate 5 clips.
- [ ] Keep the 3 best. For each: trim so the hook hits in the first 1–2 seconds,
      check captions are readable, add a 3–6 word on-screen title.
- [ ] Confirm format: 9:16 vertical, ≤ 60s, captions on.
- [ ] Post all 3 across your platforms per the campaign's submission rules.
- [ ] Add your link in bio (Gumroad product / booking page) for cross-stream pull.

## Daily loop (≈30 min)
- [ ] Post 3–5 new clips.
- [ ] Log each in the tracker below: which hook, platform, 24h views.
- [ ] Note your 2 best performers; make more like them tomorrow.

## Simple tracker (copy into a sheet)
```
date | campaign | clip_hook | platform | views_24h | views_7d | payout | notes
```

## Week-1 success = process, not payout
If you posted ~20+ clips and can name which hook style got the most views,
you're winning — the money follows the reps. Re-up the niche/campaign that
performed; drop what didn't.

## Don't
- Don't buy views/followers (voids payouts).
- Don't clip content you're not authorized to use.
- Don't spread across 6 niches — pick one, learn its hooks, then expand.
